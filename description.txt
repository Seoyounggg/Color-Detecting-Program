Color Detecting Program

1. 간단한 프로그램 설명 및 실행방법

  -  프로그램 설명
  Color Detecting Program은 드론으로부터 받은 이미지가 test_data에 있다는 전제하에 시작됩니다.
  프로그램 실행 후 드론으로부터 받은 이미지를 선택한 뒤, 이미지에서 경로의 색이 달라보이는 3개의 지점을 클릭하면
  mask_generator.py 파일이 있는 위치에 경로인 곳은 흰색 아닌 곳은 검은 색으로 구분되는 이미지 파일이 자동 저장됩니다.
  프로그램을 종료하지 않고 결과를 확인하려면, SHOW 버튼을 클릭하면 됩니다.

  -  실행방법
  1) test_data 폴더에 경로 이미지를 넣어줍니다. (이미 만들어져 있습니다)
  2) mask_generator.py를 실행하면 윈도우가 뜹니다.
  3) 윈도우 상단에 있는 파일을 클릭 후 열기를 클릭한 다음 경로 이미지를 선택합니다.
  4) 이미지를 선택하면, 윈도우의 좌측에 선택한 이미지가 뜹니다.
  5) 이미지 내에서 경로 중에서 색이 달라보이는 3개의 점을 클릭합니다.
  6) 3번의 클릭 후 윈도우의 하단에 있는 'SHOW'라는 버튼을 클릭하여 변환된 이미지를 윈도우의 우측에서 확인합니다.
     이때, 변환된 이미지는 mask_generator.py와 같은 위치에 'result_'+'클릭한 파일명' 으로 자동 저장됩니다. 
  7) 프로그램을 종료하기 위해서는 윈도우의 X 버튼을 클릭하거나,
      윈도우 상단에 있는 파일 클릭 후 종료를 클릭하면 됩니다.

  새로운 이미지로 다시 돌려볼 경우 이전의 이미지로 실행한 프로그램을 종료한 뒤 프로그램을 다시 실행시키고 위의 순서대로 진행하면 됩니다.
